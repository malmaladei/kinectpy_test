Install Visual Studio 2013 http://www.visualstudio.com/en-us/downloads
Install Pytools for Visual Studio http://pytools.codeplex.com/ 
Install Git - obviously
Install Python 2.7.6 from python.org (make sure to add it as an environment-variable in Windows and as a Python-environment in VS13 if you have multiple Pythons installed)
Install Kinect 1.0 SDK from http://www.microsoft.com/en-us/download/details.aspx?id=28782 => interacts with Kinect hardware
Install Python-Kinect wrapper Pykinect: http://pypi.python.org/pypi/pykinect/1.0 => implements Kinect functionality in Python
Download vpykinect module from https://sites.google.com/site/erfarmer/downloads => integrates with Pykinect to get input from Kinect
VPython from http://www.vpython.org/ => displays output from vpykinect

after the tutorial http://possiblywrong.wordpress.com/2012/11/04/kinect-skeleton-tracking-with-visual-python/
